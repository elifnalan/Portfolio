🧠 Default Mode Network Analysis

Python · NumPy · pandas · scikit-learn · SciPy

Data-driven analysis of Default Mode Network connectivity differences in autism using resting-state fMRI from the ABIDE dataset (871 participants).

ResearchMLNeuroscience
view research abstract ▾

This study explores alterations in brain connectivity associated with autism spectrum disorder (ASD) using resting-state functional MRI. Focusing on the Default Mode Network — a system implicated in self-referential thought and social cognition — data were drawn from the Autism Brain Imaging Data Exchange (ABIDE), yielding 871 quality-controlled participants (403 ASD, 468 controls). Sixteen DMN regions of interest were defined with the Schaefer 200-region atlas, and connectivity matrices were computed from ROI time-series correlations. Welch's t-tests revealed hypo-connectivity in ASD between medial prefrontal and posterior cingulate regions, consistent with reports of disrupted anterior–posterior DMN integration. Future analyses will apply machine learning and deep learning approaches to classify diagnostic groups and evaluate developmental subgroups.
